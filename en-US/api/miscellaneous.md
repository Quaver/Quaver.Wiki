## GET `/v1/stats`

Gets server stats

### Example Response

```json

{
    "status": 200,
    "stats": {
        "total_online_users": 96,
        "total_users": 95973,
        "total_mapsets": 4669,
        "total_scores": 5987238
    }
}
```
