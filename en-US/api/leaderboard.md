## GET `/v1/leaderboard`

Retrieves the top 50 users on the leaderboard

### Parameters

| Name    | Description       | Required |
| ------- | ----------------- | -------- |
| mode    | Game mode         | Yes      |
| country | Filter by country | No       |
| page    | Pagination        | No       |
