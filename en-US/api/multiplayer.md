## GET `/v1/multiplayer/leaderboard`

Gets the data for the multiplayer wins leaderboard

### Parameters

| Name | Description        | Required |
| ---- | ------------------ | -------- |
| mode | Key4 = 1, Key7 = 2 | Yes      |
| page | Pagination         | No       |

### Example Response

